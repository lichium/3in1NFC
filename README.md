# 3in1NFC

---

## Introduction

A 3-in-1 NFC card, controlled by a SP3T switch.

### Reason

I have so many cards and I have to bring all of them when I go out, which makes me very troubled. An 3in1 NFC card can greatly reduce the cards I have to take.
